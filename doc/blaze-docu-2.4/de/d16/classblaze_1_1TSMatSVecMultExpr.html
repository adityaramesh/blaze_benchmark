Detailed Description

template<typename MT, typename VT>
class blaze::TSMatSVecMultExpr< MT, VT >

Expression object for sparse matrix-sparse vector multiplications.

The TSMatSVecMultExpr class represents the compile time expression for multiplications between column-major sparse matrices and sparse vectors.
