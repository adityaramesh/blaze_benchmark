Detailed Description

template<typename VT1, typename VT2, bool TF>
class blaze::SVecDVecSubExpr< VT1, VT2, TF >

Expression object for sparse vector-dense vector subtractions.

The SVecDVecSubExpr class represents the compile time expression for subtractions between a sparse vector and a dense vector.
