Detailed Description

template<typename VT, typename ST, bool TF>
class blaze::SVecScalarMultExpr< VT, ST, TF >

Expression object for sparse vector-scalar multiplications.

The SVecScalarMultExpr class represents the compile time expression for multiplications between a sparse vector and a scalar value.
