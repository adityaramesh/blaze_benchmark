Detailed Description

template<typename MT1, typename MT2, bool SO>
class blaze::SMatDMatSubExpr< MT1, MT2, SO >

Expression object for sparse matrix-dense matrix subtractions.

The SMatDMatSubExpr class represents the compile time expression for subtractions between a sparse matrix and a dense matrix with identical storage order.
