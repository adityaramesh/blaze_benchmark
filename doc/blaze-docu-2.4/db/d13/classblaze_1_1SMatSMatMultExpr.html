Detailed Description

template<typename MT1, typename MT2>
class blaze::SMatSMatMultExpr< MT1, MT2 >

Expression object for sparse matrix-sparse matrix multiplications.

The SMatSMatMultExpr class represents the compile time expression for multiplications between row-major sparse matrices.
