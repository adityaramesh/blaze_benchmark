Detailed Description

template<typename VT, typename MT>
class blaze::TSVecDMatMultExpr< VT, MT >

Expression object for transpose sparse vector-dense matrix multiplications.

The TSVecDMatMultExpr class represents the compile time expression for multiplications between transpose sparse vectors and row-major dense matrices.
