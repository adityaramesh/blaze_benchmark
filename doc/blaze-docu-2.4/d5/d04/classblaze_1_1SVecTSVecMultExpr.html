template<typename VT1, typename VT2>
class blaze::SVecTSVecMultExpr< VT1, VT2 >

Expression object for sparse vector-sparse vector outer products.

The SVecTSVecMultExpr class represents the compile time expression for sparse vector-sparse vector outer products.
